Here’s a complete, runnable HTML example that demonstrates the basic structure of a webpage with headings, paragraphs, images, links, and a simple table.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First HTML Page</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 20px;
            background-color: #f4f4f4;
        }
        h1 {
            color: #333;
        }
        table {
            border-collapse: collapse;
            width: 50%;
            margin-top: 10px;
        }
        table, th, td {
            border: 1px solid #555;
        }
        th, td {
            padding: 8px;
            text-align: left;
        }
        a {
            color: blue;
            text-decoration: none;
        }
        a:hover {
            text-decoration: underline;
        }
    </style>
</head>
<body>

    <h1>Welcome to My Webpage</h1>
    <p>This is a simple HTML page example with text, images, links, and a table.</p>

    <h2>Image Example</h2>
    <img src="https://via.placeholder.com/300" alt="Sample Image" width="300">

    <h2>Link Example</h2>
    <p>Visit <a href="https://www.example.com" target="_blank">Example Website</a></p>

    <h2>Table Example</h2>
    <table>
        <tr>
            <th>Name</th>
            <th>Age</th>
        </tr>
        <tr>
            <td>Alice</td>
            <td>25</td>
        </tr>
        <tr>
            <td>Bob</td>
            <td>30</td>
        </tr>
    </table>

</body>
</html>

Features in this HTML:
<!DOCTYPE html> → Declares HTML5 document type.
<meta> tags → Set character encoding and make the page responsive.
CSS in <style> → Adds basic styling.
Headings (<h1>, <h2>) → Organize content.
Paragraphs (<p>) → Display text.
Image (<img>) → Displays an image with alt text.
Link (<a>) → Clickable hyperlink.
Table (<table>) → Displays tabular data.
